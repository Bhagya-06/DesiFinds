# LangGraph Workflow

Maintain state with TypedDict.

### Agent 1: Product Deconstructor

Extract:

* Category
* Features
* Materials
* Price range
* Aesthetic style

### Agent 2: RAG Matcher

Query ChromaDB and retrieve top 5 most relevant Indian products.

### Agent 3: Quality Curator

Evaluate similarity and generate:

* Match score
* Comparison explanation
* Value proposition
* Craftsmanship insights

### Agent 4: Formatter

Prepare structured response for frontend.
